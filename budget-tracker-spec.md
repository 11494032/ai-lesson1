1. 项目名称
极简个人记账工具 (Simple Personal Budget Tracker)
2. 项目概述
一个轻量化的个人记账工具，用户可快速记录每笔消费金额与类别，系统自动统计周 / 月支出数据，并通过直观图表展示消费趋势与结构，帮助用户清晰掌握财务状况。
3. 核心功能
3.1 基础记账功能
添加记录：用户可输入消费金额、选择消费类别（如餐饮、交通、娱乐、购物等）、填写备注（可选）、选择消费日期（默认当天）。
编辑 / 删除记录：支持对已添加的记录进行修改或删除操作。
类别管理：提供预设常用类别，同时允许用户自定义新增 / 删除消费类别。
3.2 数据统计功能
时间维度统计：
按周统计：展示本周总支出、日均支出、各分类支出占比。
按月统计：展示本月总支出、日均支出、各分类支出占比。
趋势分析：展示近 4 周 / 近 6 个月的支出变化趋势。
分类统计：展示各消费类别的总支出及占比。
3.3 数据可视化（图表展示）
饼图：展示指定时间段（周 / 月）内各消费类别的占比分布。
折线图：展示指定时间段内每日 / 每周的支出金额变化趋势。
柱状图：对比不同时间段（如本月与上月）的总支出，或展示各分类的支出金额。
3.4 数据持久化
本地存储：将记账数据保存在浏览器本地（LocalStorage）或文件中，确保刷新页面后数据不丢失。
（可选）导出功能：支持将记账数据导出为 CSV/Excel 文件，方便备份或在其他软件中打开。
4. 界面布局（Web 版本示例）
4.1 顶部导航栏
左侧：显示项目名称「极简记账」。
右侧：切换统计视图（「本周」/「本月」）、「设置」按钮（用于管理类别、导出数据）。
4.2 左侧 / 顶部：记账输入区
输入框：金额（数字输入）。
下拉选择框：消费类别。
输入框：备注（可选）。
日期选择器：选择消费日期（默认今天）。
按钮：「添加记录」。
4.3 中间：记录列表区
展示最近的消费记录列表，每条记录包含：日期、类别、金额、备注。
每条记录右侧提供「编辑」和「删除」按钮。
4.4 右侧 / 底部：数据统计与图表区
概览卡片：显示当前视图（本周 / 本月）的总支出、平均每日支出。
图表区域：
标签页切换：「分类占比」（饼图）、「支出趋势」（折线图）。
自动渲染对应图表，直观展示统计数据。
5. 技术选型建议
前端框架：HTML5 + CSS3 + JavaScript（原生或轻量框架如 Vue.js/React）
图表库：Chart.js（轻量、易用，适合展示饼图、折线图、柱状图）
数据存储：浏览器 LocalStorage（实现简单，无需后端）
部署方式：纯前端静态页面，可直接在浏览器打开，或部署到 GitHub Pages / Vercel 等平台。
6. 交互流程
用户首次打开页面，系统初始化默认类别与空数据。
用户在记账区输入金额、选择类别等信息，点击「添加记录」。
记录被保存到本地存储，同时更新下方的记录列表与右侧的统计图表。
用户点击「本周」/「本月」切换按钮，图表与统计数据自动更新为对应时间范围。
用户点击记录旁的「编辑」/「删除」按钮，修改或移除数据后，所有视图同步更新。
（可选）用户在设置页点击「导出数据」，生成并下载 CSV 文件。
